# Welcome to the documentation for the "Perovskite Tandem Database"

## Introduction
The goal of this project is to encourage a better description of perovskite tandem devices that aligns with the FAIR data principles, i.e. findable, accessible, interoperable, and reusable. The resources that have been developed to meet this end are described here.

### Data schemas for describing perovskite tandem devices
A core part of the initiative is the development of data schema and guidelines for how to describe a perovskite device. 

- A detailed description of the data schema is found in the "Data Schemas" section in the menu to the left. 

### Online resources
There is a database housed by NOMAD for perovskite data files. Associated to the database is a graphical exploration tool for filtering, visualising, and analysing perovskite tandem data files stored in the NOMAD database. Those resources are found at:

- [Perovskite Tandem Databse](https://nomad-lab.eu/prod/v1/oasis/gui/search/tandem-solar-cells-database)
